## :rocket: Sobre o desafio
A integração deve buscar as oportunidades com status igual a ganho no Pipedrive, depois inseri-las como pedido no Bling

```sh

  $ yarn install

  # Iniciar aplicação
  $ yarn dev:server
```

ROTAS
- **`GET /orders`**: LISTAR PEDIDOS

- **`POST /create`**: CRIAR PEDIDO NO BLING
